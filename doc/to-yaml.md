# uxy to-yaml

Converts UXY format to YAML.

### Example

<pre>
<b>$ uxy ps | uxy to-yaml<b>
- PID: '512'
  TIME: 00:00:00 uxy
  TTY: pts/22
- PID: '513'
  TIME: 00:00:00 uxy
  TTY: pts/22
- PID: '514'
  TIME: 00:00:00 ps
  TTY: pts/22
- CMD: bash
  PID: '12392'
  TIME: 00:00:01
  TTY: pts/22
</pre>
